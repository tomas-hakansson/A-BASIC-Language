ABL - A BASIC Language

An Altair BASIC player, written by Tomas Håkansson and Anders Hesselbom. The initial goal is to be able to run all games in the book 101 BASIC Computer Games (1975).

Control commands

Control commands can be used in immediate and are used to configure the ABL application. Control commands are not case-sensitive.

!FULLSCREEN

Controls if the application is displayed in screen mode.

!FULLSCREEN

Toggles fullscreen mode.

!FULLSCREEN ON

Sets the application to fullscreen mode.

!FULLSCREEN OFF

Sets the application to windowed mode.

!LOG

Controls if the debug output is rendered on screen.

!LOG

Toggles debug output rendering.

!LOG ON

Turns on debug output rendering.

!LOG OFF

Turns off debug output rendering.

!QUIT

Closes the application immediately.

!RES

Controls the graphical resolution of the ABL application. The supported modes are:

  • 640 * 200 pixels (80 * 25 characters)
  • 480 * 200 pixels (60 * 25 characters) - default
  • 320 * 200 pixels (40 * 25 characters)
  • 160 * 200 pixels (20 * 25 characters)

!RES

Toggles the current resolution from the current to the next in the list. The first time !RES is used, the resolution will change from 480 * 200 pixels (60 * 25 characters) to 320 * 200 pixels (40 * 25 characters). The second time !RES is used, the resolution will change to 160 * 200 pixels (20 * 25 characters), and so on.

!RES 1

Sets the resolution to 640 * 200 pixels (80 * 25 characters), the highest resolution.

!RES 2

Sets the resolution to 480 * 200 pixels (60 * 25 characters), the default resolution.

!RES 3

Sets the resolution to 320 * 200 pixels (40 * 25 characters), the resolution used in the Commodore 64.

!RES 4

Sets the resolution to 160 * 200 pixels (20 * 25 characters), for high speed graphics. The C64 can display multicolor pictures in this resolution.
